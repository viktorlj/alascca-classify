Getting started

Installation & CLI

Install alascca-classify locally and use it from the command line.

Requirements

  • Python ≥ 3.11
  • uv package manager (recommended) or pip
  • Git

Installation

Option 1 — From GitHub (recommended)

git clone https://github.com/viktorlj/alascca-classify.git
cd alascca-classify
uv venv && source .venv/bin/activate
uv sync

Option 2 — Direct install with uv

uv pip install git+https://github.com/viktorlj/alascca-classify.git

After installation, the alascca-classify command is available in your shell.

CLI commands

classify

Classify a single sample from a MAF file.

# JSON output (default)
alascca-classify classify sample.maf

# With SEG file for PTEN deep deletion detection
alascca-classify classify sample.maf --seg sample.seg

# Human-readable text output
alascca-classify classify sample.maf --human

# Generate PDF report
alascca-classify classify sample.maf --pdf -o report.pdf

# With sample metadata
alascca-classify classify sample.maf --sample-id PATIENT_001 --metadata meta.json
OptionDescription
--segPath to SEG file for PTEN deletion detection
--metadataPath to JSON metadata file (sample_id, ptnm_stage, tumor_location, etc.)
--sample-idSample identifier (overrides metadata file)
-o, --outputOutput file path (prints to stdout if omitted)
--humanHuman-readable text output
--pdfGenerate PDF report

classify-batch

Classify a directory of MAF files in one go.

# Basic batch classification
alascca-classify classify-batch --input-dir ./samples/ --output results.tsv

# With SEG files and metadata
alascca-classify classify-batch --input-dir ./samples/ --seg-dir ./segments/ \
  --metadata metadata.tsv --output results.tsv
OptionDescription
--input-dirDirectory containing .maf files (required)
--seg-dirDirectory containing .seg files (matched by sample name)
--metadataTSV file with sample_id column and additional metadata
-o, --outputOutput TSV file (default: results.tsv)

check-variant

Quickly check how a specific variant is classified.

# Check a PIK3CA hotspot
alascca-classify check-variant PIK3CA "p.E545K"

# Check a PTEN nonsense variant
alascca-classify check-variant PTEN "p.R130*" --type Nonsense_Mutation

# Check a PIK3R1 missense
alascca-classify check-variant PIK3R1 "p.N564D"
Argument / OptionDescription
GENEGene symbol: PIK3CA, PIK3R1, or PTEN
HGVSPProtein change, e.g. p.E545K
--typeMAF Variant_Classification (default: Missense_Mutation)

serve

Start the web interface locally.

alascca-classify serve --port 8000
# Open http://localhost:8000
OptionDescription
-p, --portPort to serve on (default: 8000)
--hostHost to bind to (default: 127.0.0.1)

version

Print the installed version.

alascca-classify version

Input formats

MAF file

Tab-separated, minimum required columns:

ColumnRequiredDescription
Hugo_SymbolYesGene symbol (PIK3CA, PIK3R1, PTEN, etc.)
Variant_ClassificationYesVEP consequence (Missense_Mutation, Frame_Shift_Del, etc.)
HGVSp_ShortYesProtein change (e.g. p.E545K)
t_alt_countNoTumor alt read count (for VAF calculation)
t_ref_countNoTumor ref read count (for VAF calculation)

SEG file

Tab-separated, for PTEN deep deletion detection:

ColumnDescription
IDSample identifier
chromChromosome
loc.startSegment start position
loc.endSegment end position
num.markNumber of probes in segment
seg.meanLog2 copy ratio

Output

The classify command produces JSON by default with fields including alascca_group (A, B, or none), aspirin_eligible (boolean), classified variant details, informational biomarkers (BRAF, KRAS, NRAS, MSI), and clinical notes.

The classify-batch command produces a TSV summary with one row per sample.

Source code

The source code, demo files, and full documentation are available on GitHub. Bug reports and contributions are welcome.
