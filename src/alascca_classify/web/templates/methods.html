Methods

Classification algorithm

Based on the ALASCCA trial (Martling et al., NEJM 2025;393:1051-64) and supplementary Table S3.

The ALASCCA trial

ALASCCA was a double-blind, randomized, placebo-controlled trial of adjuvant aspirin (160 mg/day for 3 years) in resected stage I–III colorectal cancer with somatic PI3K pathway alterations. Key findings:

  • Group A (PIK3CA exon 9/20 hotspots): 3-year recurrence 7.7% aspirin vs 14.1% placebo (HR 0.49, p=0.04)
  • Group B (other PI3K pathway alterations): 3-year recurrence 7.7% aspirin vs 16.8% placebo (HR 0.42)
  • Combined: HR 0.45 (95% CI 0.28–0.74)
  • ~37% of CRC patients have PI3K pathway alterations (17.3% Group A, 19.7% Group B)

Group B — Other PI3K pathway alterations

Priority rule

If a patient has both Group A and Group B alterations, they are assigned to Group A.

PTEN deep deletion

From the ALASCCA supplementary methods:

  • Requires estimated tumor cell fraction ≥ 0.30
  • A deletion is called if a CNV segment overlapping PTEN (including at least one exon) shows at least 15% lower DNA abundance (absolute log-ratio difference of 0.2) compared to flanking regions both 3 Mb upstream and downstream of PTEN
  • This is the only CNV-based alteration in the classification

Classification algorithm

1
Parse MAF file → extract variants in PIK3CA, PIK3R1, PTEN
2
Check PIK3CA variants against Group A hotspot positions (7 positions in exon 9/20)
3
Check remaining PI3K pathway variants against Group B consequence types
4
If SEG file provided: assess PTEN deep deletion from copy number data
5
Apply priority: Group A > Group B > none
6
Detect informational biomarkers: BRAF V600E, KRAS, NRAS, MSI status

Caveats

  1. Research and clinical decision support only. Final treatment decisions must be made by the treating physician.
  2. PTEN deep deletion requires adequate tumor purity (≥30%) and uses a simplified algorithm.
  3. The tool classifies variants but does not assess pathogenicity. All missense variants are included per ALASCCA protocol (VEP moderate impact threshold).
  4. Germline filtering is assumed. The input MAF should contain somatic variants only.
  5. This tool does not replace the full ALASCCA genomic pipeline which included custom library prep and manual QC review.
